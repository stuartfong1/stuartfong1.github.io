Projects

Bloo

This was my final project in my grade 11 computer science course that I worked on with a partner. It is a platformer game which was created in Pygame, which contains a currency system, enemies, and a storyline. It has a total of two levels, as well as a boss battle.

Autonomous Robot

This is a project that I am currently working on with my coworkers at Robo-Geek Inc. It is a robot that is uses line detection to navigate along a path. It uses a Raspberry Pi to get the camera input, an Arduino to control the motors for the wheels, and Serial Peripheral Interface (SPI) to communicate between the two. It also uses Robotic Operating System (ROS) to view the robot's camera from a remote laptop.

Sorting Algorithms

This is an interactive visualization of various sorting algorithms made using Processing. The user is able to input values or randomize them, and is then able to see the order of the values as they are being sorted.

AVL Tree

This is an interactive visualization of an AVL Tree, a type of balanced binary search tree that was made using Processing. It is able to insert, delete, and find nodes, all while balancing the branches of the tree. This keeps operations in O(log n) time.

Self-Playing Video Game

This a program that was part of the NEXUS program offered @ Mathstronauts. It features a rocket that is able to navigate its way around falling asteroids. It uses a type of reinforcement learning called Q-learning to learn how to avoid the asteroids using prior experience.

Udacity Tensorflow Course

I took this course as a part of Google Code-in 2019, which sparked my interest in machine learning. I learned topics such as image recognition, stock forecasting, and natural language processing.

Searching Algorithms

This is a visualization of various searching algorithms made using Processing. The user is able to build obstacles between the start and end points, and is then able to choose from various algorithms to construct a path between the two points.

Linear Regression

This is a linear regression program that I made using Tensorflow that is able to approximate the line of best fit of a scatterplot. It uses a neural network and mean squared error to determine the equation of the line of best fit.
